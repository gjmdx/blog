Jade 6.5 图文安装教程

发表于 2017-11-23 更新于 2020-03-20 分类于 教程
本文字数： 1.3k 阅读时长 ≈ 1 分钟

发布日期

2017-11-23 15:23

制作教程时的环境

虚拟机：Parallels Desktop 13
操作系统环境：基于 msdn 完全纯净的 win10 专业版（1703）

名词解释

看清楚我让你去哪里找什么，不要弄错了。

  1. 你下载下来并解压的文件，以下称为”安装包”。
  2. 安装时让你选择的路径，以下称为”安装目录”。

账户名查看方法

打开任务管理器，找到”用户”，这就是你的账户名（务必注意大小写和特殊符号），本步骤下面会用到。

安装前准备工作

  1. 关闭所有安全软件以防在修改注册表时被拒绝。
  2. 确保目前登陆账户不是 Administrator（上面说了怎么查看账户名）。
  3. 将安装文件拷贝至不含任何中文及特殊符号的目录中。
  4. 下载后务必将安装包解压！

安装主程序

  1. 安装包解压后如图所示，双击圆圈中的文件安装。

  2. 选择安装路径，点击安装，直至安装完成。安装结束后，窗口会自动关闭。

注册 OCX 控件

安装结束后，软件是无法正常运行的，此时我们要注册对象链接和嵌入用户控件（OCX）。

  1. 在安装包中找到”ocx”文件夹，右键选择”注册 ocx.bat”，找到”编辑”（如果 win7 用户没有这个选项的话，就用记事本打开。）

  2. 将第一行修改为安装盘符（安装路径前 2 个字符，如 C:），第二行修改为”cd 安装路径”，如图所示，然后保存。

  3. 右键选择”注册 ocx.bat”，以管理员模式运行，接着会提示多个”xxxx 已成功”，全部点击确定直到窗口关闭。

提示，在进行以上步骤时，如果 grid32.ocx 和 MSCOMCTL.OCX 安装报错，请将安装包中所有的.ocx 文件拷贝至 C:\Windows\SysWOW64（假设系统盘是 C 盘）后，重做第二、三步。
此时，将第一行修改为安装盘符 C:，第二行修改为 cd C:\Windows\SysWOW64，然后保存，接着以管理员模式运行。

添加注册表

本步骤务必注意大小写。

  1. 使用”win+R”组合键打开”运行”，键入以下命令，打开注册表编辑器。

    %systemroot%/regedit
    (32 位系统）
    %systemroot%/syswow64/regedit
    (64 位系统）

  2. 展开”HKEY_LOCAL_MACHINE”，展开”SOFTWARE”，右键”SOFTWARE”，选择”添加项”，命名为”MDI”。

  3. 在”MDI”下新建一个新项，名为”license”。

  4. 在”license”左侧新建一个”字符串值”，命名为”Owner”。

  5. 修改”Owner”的值为你的用户名。

配置 PDF 卡片

  1. 随便打开一个示例文件（打开你自己的也行），点击上方”pdf”，选择”setup”。

  2. 点击如图所示按钮，在安装包中找到”pdf2.dat”文件，这个文件在安装包的”jade65/PDF22004”中。

  3. 选定后，点击”select all”，然后”create”。

  4. 由于本步骤时间很长，开始时会提示如下窗口。
    · 询问是否在完成后关机，选择否。
    · 询问是否在完成后注销，选择否。

  5. 全部完成后，将提示：

    已知部分童鞋在载入 pdf 卡片时出现 Run-time error ‘75’的报错，原因是没有管理员权限，无法写入相关数据。解决方法是，以管理员模式启动 Jade（以后不再需要，此问题多发于 win7 以上系统）。

完成安装

按所学方法进行查询，如果可以正常查找，则安装成功。
